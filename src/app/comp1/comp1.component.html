RxJs Creation Operators Demo

This app demonstrates use of 5 creation operators :

  • timer : Completes after set time.
  • interval : Never ends, emits recurringly every time inteval set.

App description

Each creation operator creates an Observable.

Each of these , when subscribed to, will emit its values.

The user will trigger each subscription with a different button.

The output will be displayed in the dev-tools browser console.

App Controls

Started, will complete in {{ timerLengthMilisec / 1000 }} seconds.

Current count is : {{ intervalCounter }} will end after {{ timerLengthMilisec / 1000 }} seconds.
