Vanilla Observable

This app demonstrates raw use of Observables & subscription.

The subscription function does the following things :

  • Emits a syncronous value, this means it immediatly emmits.
  • Emits an asyncronous value, this means it waits some time before it emmits, while the runtime continues in its flow.
  • It either ends with a COMPLETE or an ERROR
  • In all cases it runs a teardown functin (just a message to console log)

You control the async wait time and also how the subscription ends.

The variables whose values you control are:

  • Timeout for the asynchronous (it' sthe 2nd) emission asyncEmissionTimeMillisec = {{ asyncEmissionTimeMillisec }} milsec.
  • Value of isActivateComplete = true so ENDING USING COMPLETE
  • Value of isActivateComplete = false so ENDING USING ERROR
    •

The runtime results are displayed in the Dev Tools console.
