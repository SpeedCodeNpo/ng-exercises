Observable Demo

This app demonstrates raw use of an Observable & subscribing to it.

App description

The subscription function does the following things :

  • Emits a syncronous value, this means it immediatly emmits.
  • Emits two asyncronous value, these are emitting after a time interval.
  • Depending on values set in code, the Observable might end with a COMPLETE , an ERROR or not end.
  • In case it has no ending (no COMPLETE or ERROR emitted) then the only way to stop a subscription is to unsubscribe.
  • In all cases (whether it ends with a COMPLETE, ERROR or by unsubscribe) the Observable's Teardown function is executed (it will console.log a teardown message).

Controlling the app

You control the async wait time and also how the subscription ends.

The variables whose values you control are:

  • asyncEmissionTimeMillisec = {{ asyncEmissionTimeMillisec }} milsec , this is the timeout for the first async emission. The 2nd async emission is 3 times that value = {{ 3 * asyncEmissionTimeMillisec }}

  • isNeverEnding == {{ isNeverEnding }} , this means that the Observable wont end, if subscribed then the only way to free the subscription is using unsubscribe.

  • isActivateComplete = true , this means ending the Observable using 'COMPLETE' function.
  • isActivateComplete = false , this means ending the Observable using 'ERROR' function.

The runtime results are displayed in the Dev Tools console.
