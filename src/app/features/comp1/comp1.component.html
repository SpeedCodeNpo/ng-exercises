signal COMPUTE

When either one of the signals ( count() or userInfo().age ) is changed, then the 'tokens' compute is activated, tokens = age * count.

So there is no button, just see this result : 'tokens' = {{ tokens() }}

signal EFFECT

There are two effects defined in the component's constructor.

One is for the count() and the other for userInfo().

When either one changes it triggers a console log for that specific one.

Open the console to see the logs.
